# Process Folder — Monthly Reports

> โฟลเดอร์นี้เก็บ **Monthly Process Report** รายงานประจำเดือนของโปรเจกต์ KU HOME API

## Naming Convention

ไฟล์ตั้งชื่อตาม format: **`DD_M_YY.md`**

| ส่วน | ความหมาย | ตัวอย่าง |
|---|---|---|
| `DD` | วันที่รายงาน (16 ของทุกเดือน) | `16` |
| `M` | เดือน (1-12) | `5` = พฤษภาคม |
| `YY` | ปี (2 หลัก) | `26` = 2026 |

### ตัวอย่าง
- `16_5_26.md` — รายงานประจำเดือนพฤษภาคม 2026
- `16_6_26.md` — รายงานประจำเดือนมิถุนายน 2026

## เนื้อหาในแต่ละ Report

แต่ละไฟล์จะบันทึก:
- **Test Suite** ทั้งหมดที่รันผ่าน (จำนวน tests + assertions)
- **Issues ที่แก้ไขแล้ว** — ปัญหา, วิธีแก้, ไฟล์ที่เกี่ยวข้อง, tests ที่ verify
- **Bug fixes** ที่พบระหว่างพัฒนา
- **เปลี่ยนแปลงสำคัญ** อื่นๆ

## Schedule

รายงานจะถูกสร้างทุกวันที่ **16 ของเดือน** หรือเมื่อมีการ update สำคัญ

# 📋 Process Report — 19 มิถุนายน 2026

## 🎯 สรุปการเปลี่ยนแปลง

### ปัญหา (Issue #20)
`StoreBookingRequest` ยอมรับ `breakfast_price`, `early_checkIn_price`, `late_checkOut_price` จาก client โดยตรง → client สามารถส่ง `breakfast_price: 0` เพื่อได้ breakfast ฟรีได้

### วิธีแก้
**แยก default prices ออกเป็นตารางใหม่ `addon_rates`** — server ดึงราคาเองจาก database, ไม่ trust client

คุณท่านเลือก approach นี้เพราะ:
- Static default values ที่เปลี่ยนได้ = ควรเป็นข้อมูลใน DB
- Add-on price ไม่จำเป็นต้องเก็บใน `booking` หรือ `booking_room` — ใช้ lookup จาก `addon_rates` ตอนคำนวณ `total_amount` พอ
- ในอนาคตถ้าจะ snapshot ราคาตอนจอง สามารถเพิ่ม column ใน `addons` หรือ `booking_rooms` ได้ทีหลัง
